# 에피소드 사례집

에피소드 발굴 시 참고할 수 있는 카테고리별 사례 모음.
실제 합격 자소서에서 발췌·각색한 에피소드 구조를 정리한다.

## 목차

1. 약한 관점 vs 좋은 관점
2. 도전/극복 에피소드
3. 팀워크/리더십 에피소드
4. 창의적 문제해결 에피소드

## 1. 약한 관점 vs 좋은 관점

### 자주 버려야 하는 약한 관점

- **당연한 일을 했을 뿐인 관점**: 원래 해야 하는 일을 수행한 수준에 머무는 경우
- **조연이 너무 흔한 관점**: 누구나 비슷하게 쓸 수 있는 구도만 남는 경우
- **성과만 남고 과정이 비는 관점**: 결과는 있는데 문제 인식과 액션이 흐린 경우
- **제안형 클리셰**: 메뉴 제안, 진열 변경처럼 흔한 구조인데 구체 근거가 없는 경우

### 좋은 관점으로 바꾸는 방법

- **매출** 대신 **결품·객단가·이익률·운영 효율**로 바꾼다
- **외부 변수** 대신 **내부 프로세스의 비효율**을 본다
- **문제 발생 후 해결** 대신 **내가 먼저 문제를 제기한 경험**을 찾는다
- **갈등 해소** 대신 **팀이 제대로 토론하지 못한 구조를 바꾼 경험**을 본다

### 빠른 판정 체크

- "다른 지원자도 비슷하게 쓸 수 있는가?"
- "내 행동을 HOW까지 2~3개 설명할 수 있는가?"
- "직무에서 실제로 할 법한 문제와 연결되는가?"
- "성과가 약해도 판단과 문제 인식은 선명한가?"

## 2. 도전/극복 에피소드

### 사례 1-1: 단계별 공포 극복

- **상황**: 슬럼프 → 고소공포증 극복을 목표로 설정
- **차별화 포인트**: 산 등반이라는 소재 자체가 아니라, **1차→2차 중간 목표 설정 + 이미지 트레이닝**이라는 체계적 전략
- **활용 문항**: 도전 경험, 목표 설정 및 달성

### 사례 1-2: 교환학생 학점 4.0 달성

- **상황**: "한국 교환학생은 놀러 왔다"는 편견 → 학습 로드맵 구축
- **차별화 포인트**: **정보 수집 전략**(기출 탐색, TA 활용) + **약점 인정**(Writing 첨삭 요청)
- **활용 문항**: 열정적 몰입, 자기 객관화, 전략적 사고

## 3. 팀워크/리더십 에피소드

### 사례 2-1: 축구팀 창단 — 공정한 기회

- **상황**: 4개 팀에서 출전 기회 없는 후보 선수들을 모아 팀 창단
- **차별화 포인트**: **개인 강점 기반 12개 전술** + **4쿼터제로 전원 출전** + **양방향 피드백**
- **활용 문항**: 리더십, 조직 기여, 공정성

### 사례 2-2: 발표 공포증 팀원 대신 발표

- **상황**: 발표 전날 밤 10시, 팀원이 무대 공포증 고백
- **차별화 포인트**: 원망 대신 **역할 재분배** → 팀원은 대본 피드백 → **"대본 가르쳐준 조는 처음"이라는 심사평**
- **활용 문항**: 팀워크, 위기 대응, 배려

### 사례 2-3: 교수님과의 관계 구축

- **상황**: 젊은 지도교수님이 동아리에서 위축 → 회장으로서 교수님 배려
- **차별화 포인트**: **화분 선물 같은 소소한 배려** + 행사 기획 시 의견 반영 → 교수님이 기업 컨택까지 주도
- **활용 문항**: 조직 융화, 관계 구축

### 팀워크 하위 유형 체크리스트

- **공감 후 차선책 제시형**: 상대 의견에 먼저 공감한 뒤, 더 나은 대안을 제시
- **이해관계자 조율형**: 팀원 외에도 거래처, 외부 파트너, 타 부서까지 조율
- **조직 분위기 개선형**: 갈등 원인을 개인이 아니라 구조/문화 문제로 보고 손봄
- **주파수 맞추기형**: 상사·현업·고객 성향을 파악해 보고/협업 방식을 조정
- **동기부여형**: 지친 팀을 다시 움직이게 하는 언어·보상·계획을 설계
- **역할 재분배형**: 누가 무엇을 잘하는지 기준으로 역할을 다시 짬

## 4. 창의적 문제해결 에피소드

### 사례 3-1: 편의점 결품 관리

- **상황**: 인기 도시락 결품 반복 → 점장님도 재고 리스크 경험으로 발주 망설임
- **차별화 포인트**: **결품 = 판매 기회 손실**이라는 관점(영업관리 직무 연결) + 점진적 발주량 증가 제안 + 수요 근거 제시
- **핵심**: 매출 향상(진부) 대신 **결품 최소화(차별화)**라는 관점 선택

### 사례 3-2: 삼계탕 → 닭죽 변환

- **상황**: 농촌봉사, 20인분 삼계탕에 더 많은 손님 방문
- **차별화 포인트**: 추가 구매(양적 해결) 대신 **형태 변환(질적 해결)** — 예산/시간/불확실성 3가지 제약 동시 해결
- **TRIZ 원리**: 매개변수 변화, 차원변경

### 사례 3-3: 횟집 발주 최적화

- **상황**: 매일 왕복 3시간 수산시장 + 수족관 빽빽 → 기름값 + 신선도 손실
- **차별화 포인트**: 한 달간 종류별 주문량 기록 → **적정 발주량 산정** → 주문 7회→3회 감소
- **핵심**: 경험+데이터 기반 의사결정

### 사례 3-4: 역발상 쇼핑몰 문구

- **상황**: 인터넷 쇼핑몰 조회수는 높은데 구매 전환 저조
- **차별화 포인트**: "신중하게, 충동구매 하지 마세요" **역발상 문구** 삽입
- **TRIZ 원리**: 반대로(Inversion)

## 5. 가치관/인간미 에피소드

### 사례 4-1: 할머니 집 직접 건축

- **상황**: 장마철마다 무너질 위기의 기와집 → 아버지와 직접 재건축
- **차별화 포인트**: 못 박기도 서툴렀지만 6개월 공사 → **"받은 사랑을 되돌려줄 때 기쁨"**이라는 깨달음
- **활용 문항**: 중요한 사건, 가치관

### 사례 4-2: 산 쓰레기 줍기 캠페인

- **상황**: 등산로 쓰레기 → 플래카드/민원은 효과 없음 → 직접 눈에 띄게 줍기
- **차별화 포인트**: 10~12시 등산객 피크 시간 파악 → **행동으로 인식 변화** 유도
- **핵심**: 문제 인식 → 관찰 → 실행이라는 구조

### 사례 4-3: 에이스 복학생 프로젝트

- **상황**: 2년간 아웃사이더 → 복학 후 "후배가 먼저 알아보는 선배" 목표
- **차별화 포인트**: 맨 앞자리 + 매 수업 질문 1개 준비 + SCI 논문 기반 보고서 + 후배 스터디 운영
- **활용 문항**: 도전, 자기 계발, 리더십

## 6. 관행 개선 에피소드

### 사례 5-1: 탕비실 얼음 관행 개선

- **상황**: 정수기 얼음을 바닥에 떨어뜨려도 안 줍는 관행
- **차별화 포인트**: 위생/안전/외부 인사 신뢰도 영향 → 현장 사진 + 경영지원팀 동의 → 전사 공지
- **핵심**: 사소하지만 **조직 차원 영향을 분석**하여 개선

### 사례 5-2: 실험과목 오류 관행 개선

- **상황**: 학부 때 몰랐던 VLE/LLE 실험 오류를 대학원에서 발견
- **차별화 포인트**: 조교장에게 직접 문제 제기 → 기존 관행 수정 주도
- **핵심**: **지식 깊이 + 주도적 문제 제기**

## 7. 사례 활용 시 주의사항

1. **사례를 그대로 베끼지 말 것** — 구조와 관점만 참고
2. **본인 경험으로 치환** — 동일한 구조를 본인 소재에 적용
3. **직무와 연결** — 에피소드의 핵심 역량이 지원 직무와 어떻게 연결되는지 반드시 기술
4. **구체성 확보** — 수치, 기간, 인원 등 디테일이 있어야 신뢰감 형성
5. **약한 관점은 빨리 버릴 것** — 소재보다 관점이 약한 경우가 많으므로 관점 전환을 먼저 시도
