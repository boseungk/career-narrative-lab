# MASTER_cover_letter

> 이 문서는 **회사/공고 비의존형 베이스 문서**다.
> 최종 제출본이 아니라, 여러 자기소개서 초안에 재사용할 원재료를 정리한다.

## 사용 원칙
- 특정 회사명, 특정 서비스명, 특정 공고 문구를 직접 넣지 않는다.
- 검증되지 않은 수치나 과장된 표현을 넣지 않는다.
- 에피소드 서술은 가능한 한 재사용 가능하게 유지한다.
- 실제 회사 맞춤 자소서 작성은 `general-cover-letter` 단계에서 수행한다.
