# 자기소개서 작성 스킬 (범용)

자기소개서를 작성, 검토, 수정할 때 이 스킬의 지침을 따른다.
AI는 본 스킬을 바탕으로 자소서를 출력할 때 **'전문적인 채용 컨설턴트(Expert HR Consultant)' 페르소나**를 유지해야 한다. 기계적이고 뻣뻣한 번역투, 장황한 설명, 불필요한 접속사를 철저히 배제하고, 지원자의 역량과 마인드셋을 진정성 있고 담백하게 어필하는 능동태 단문 위주로 결과물을 생성한다. 과장된 홍보 문구(광고 카피 등)는 피한다.

---

## 작성 절차

### Step 1: 사전 준비

1. **지원자 정보 파악**
    - 경력/프로젝트 문서가 있으면 탐색하여 기술 스택, 주요 성과, 역할 범위를 파악한다
    - 지원자가 **직접 수행한 작업만** 소재로 사용한다
    - 정량 수치(매출, 성과 지표 등)는 반드시 지원자에게 사실 여부를 확인한다

2. **기업 분석**
    - 인재상, 핵심 가치, 직무 요구사항을 웹 검색 또는 제공 자료로 파악한다
    - 기업의 미션과 지원자의 직업관을 연결하는 접점을 도출한다
    - 기업 호칭을 반드시 확인한다 (입사 / 입행 / 입회 등)
    - **신년사·CEO 인사말**에서 기업 방향성 키워드를 1~2개 차용하면 차별화에 효과적

### Step 2: 직무 이해도 사전 점검
자소서의 최우선 평가 요소는 직무 적합성이다. 초안 작성 전에 아래 5가지를 반드시 정리한다:
1. 해당 직무의 **핵심 목표(KPI)**
2. **핵심 업무 영역**의 구체적 범위
3. **올바른 직무 수행 방법론**
4. **직무 수행 시 부딪힐 수 있는 어려운 상황 2가지** → 본인 에피소드 중 유사 사례와 매칭
5. **이 어려움을 극복할 수 있는 나의 강점 2가지**: 에피소드에서 드러낼 핵심 역량 도출

### Step 3: 문항 분석

초안을 쓰기 전에, 각 문항이 실제로 평가하려는 역량을 파악한다:

1. **숨은 의도 파악**: 문항의 표면적 질문 너머에 채용담당자가 진짜로 확인하고 싶은 역량을 정의한다.
   - "지원 동기" → 기업 이해도 + 직업관의 진정성
   - "노력 경험" → 끈기·성장 가능성·어려움에 대한 태도
   - "강점/약점" → 자기 객관화 + 개선 의지
   - "직무 역량" → 즉전 투입 가능성·전문성의 깊이
   - "협업/갈등" → 조직 융화력·리더십 스타일
   - "성장과정" → 자질·태도·가치관 (교육으로 변화시키기 어려운 영역)
   - "직업가치관" → 가치관의 진정성 + 기업 문화 부합도

2. **문항 유형 식별**: 문항은 크게 7가지 유형으로 나뉘며, 유형에 따라 접근법이 다르다:
   - **과거 경험 서술형** (에피소드) → STAR 구조
   - **논리 서술형** (지원동기) → 가치관/경험 + 기업 연결
   - **미래 지향 서술형** (입사 후 포부) → WHAT/HOW/WHY
   - **성장·인성 서술형** (성장과정) → 가치관 형성 스토리
   - **직업가치관 서술형** → 가치관 선언 + 형성 계기 + 기업 연결
   - **갈등 해결 서술형** → 갈등 소재 6유형 + 본인만의 해결법
   - **논술형/약술형** (시사 이슈 찬반) → 메시지 한 문장 → 근거 전개

3. **문항 구조화 분해** (장문 문항 대응):
   - 긴 문항은 질문 단위로 쪼갠 뒤, 각 질문에 대한 답변을 간략히 메모한다
   - 특히 공공기관은 문항이 매우 구체적이므로, 요구사항을 하나라도 누락하면 감점

4. **소재 배정**: 평가 의도에 가장 부합하는 경험 소재를 연결한다.

5. **서술 방향 결정**: 같은 경험이라도 문항의 평가 의도에 따라 강조점이 달라진다.

### Step 4: 초안 작성

문항 유형별 상세 전략은 [question-strategies.md](question-strategies.md)를 참조한다.

### Step 5: 사실관계 검증

[checklist.md](checklist.md)의 항목을 모든 문항에 적용한다.

### Step 6: 표현 다듬기

[style-guide.md](style-guide.md)의 톤/매너, 용어 수위, 피해야 할 표현을 적용한다.

### Step 7: 소재 분배 검증 (다수 기업 지원 시)

- 동일 에피소드를 3개 이상 기업에서 동일 역할로 사용하지 않는다
- 동일 기업 내 문항 간 소재 중복을 최소화한다

### Step 8: 글자 수 점검

- byte 제한인 경우: 한글 1자 ≈ 2byte (EUC-KR 기반 채용사이트 기준)
- 초과 시 수식어·부사·중복 수식어부터 삭제한다

## 핵심 규칙

### 톤 & 매너
- **신입**: 유연함과 학습 태도. 성과는 당당하게, 팀워크와 배움 강조
- **경력**: 성과 중심, 객관적 기여도와 임팩트(숫자, 개선율) 최전선
- **공통**: 감정적 단어 배제, 객관적 행동과 숫자로 열정을 '증명'
- **합격 자소서의 공통점**: 솔직담백함, 본인만의 워딩과 깊은 성찰

### 면접 대비 원칙
자소서의 모든 문장은 면접에서 꼬리 질문의 소재가 된다. 아래를 전제로 작성한다:
- **모든 수치·성과에 대해**: "이 수치가 어떤 환경에서, 어떻게 측정된 건가요?" 질문에 답할 준비가 되어야 한다
- **모든 행동 서술에 대해**: "왜 그 방법을 선택했나요?", "다른 방법은 고려하지 않았나요?" 질문을 예상해야 한다
- **검증 불가능한 내용 금지**: 면접관이 반박할 수 있는 과장·가공된 내용은 절대 쓰지 않는다
- **꼬리 질문 유발 포인트 사전 식별**: 초안 작성 후, 각 문항에서 면접관이 가장 파고들 만한 문장 1~2개를 찾아 답변을 준비한다

### 단점 작성 원칙

[weakness-catalog.md](weakness-catalog.md)의 40가지 예시를 참조하되, 8대 원칙을 따른다:
1. 순수 단점 피하기 — 2. 양면성 활용 — 3. 이력과 정합 — 4. 단점답게
5. 장점에서 연결 — 6. 직무 고려 — 7. 기업 문화 고려 — 8. 개선 방안 필수

### 자주 하는 실수
1. 가공된 수치 (면접에서 반박 리스크)
2. 미수행 경험 서술
3. 팀 프로젝트를 1인 프로젝트처럼 묘사
4. 기업명/호칭 오류
5. 소재 중복
6. 문항 요구사항 누락
7. **행동만 열거하고 태도/마인드가 안 보이는 서술** (이공계 특히 주의)
8. **지식만 나열** — 지식+기술+태도를 한 세트로
