# Career Narrative Lab

자기소개서 작성을 위한 `.agent` 스킬 모음 저장소입니다.

## 포함된 스킬

- `general-cover-letter`  
  자기소개서 작성/수정 시 문항 분석, STAR 구조, 톤앤매너, 문장 다듬기 가이드
- `review-cover-letter`  
  작성된 초안을 체계적으로 리뷰하는 점검 스킬
- `episode-mining`  
  자기소개서용 에피소드를 발굴/정리/분류하는 스킬

## 구조

```text
.agent/
  skills/
    cover-letter-general/
    cover-letter-review/
    episode-mining/
```

## 사용 방법

Codex 환경에서 이 저장소의 `.agent` 폴더를 기준으로 스킬을 불러 사용합니다.

예시 호출:

- 자기소개서 초안 작성
- 자기소개서 초안 리뷰
- 프로젝트 기반 에피소드 발굴

## 목적

실제 경험 기반으로 설득력 있는 커리어 내러티브를 만들고,
문항 의도에 맞는 자기소개서를 빠르게 작성/개선하는 것을 목표로 합니다.
